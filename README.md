# Minimal, TypeScript/Node.js with Vitest 🍦

To use: 
### Via clone:
1. clone the repository
2. `$ cd template-node-ts && rm -rf .git` to remove .git for this template repo


### Via 'Use this template':
1. Click the 'Use this template' then `pull` it local


### On your machine:
1. `$ npm install`
2. Make cool stuff!
